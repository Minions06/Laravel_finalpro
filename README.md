# Laravel Product Showcase

## 📌 Project Description
This project is a simple product showcase system built using Laravel framework. It demonstrates MVC architecture and full CRUD functionality (Create, Read, Update, Delete) with database integration.

---

## ⚙️ Features
- Add Product
- View Products
- Update Product
- Delete Product
- Upload Product Image
- Where to Buy field

---

## 🛠️ Tech Stack
- Laravel
- PHP
- MySQL
- Bootstrap

---

## 🚀 How to Run Locally

### 1. Clone Repository
git clone https://github.com/Minions06/Laravel_finalpro.git

### 2. Install Dependencies
composer install

### 3. Setup Environment

Copy `.env.example` to `.env`

Update database:

DB_CONNECTION=mysql  
DB_HOST=127.0.0.1  
DB_PORT=3306  
DB_DATABASE=mylaravel_web  
DB_USERNAME=root  
DB_PASSWORD=

### 4. Generate App Key
php artisan key:generate

### 5. Run Migration
php artisan migrate

### 6. (Optional) Seed Database
php artisan db:seed

### 7. Run Server
php artisan serve
